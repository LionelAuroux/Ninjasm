# Ninjasm

Python + ASM for Ninja

Ninjasm is a Python module and a template preprocessor for Assembly language.

## Platform

The actual version support:

- Linux operating system

## Assembly

The actual version support:

- nasm syntax

## Architecture

The actual version support:

- x86_64

## Installation

```bash
    git clone https://github.com/LionelAuroux/Ninjasm.git
    cd Ninjasm
    python3 -m build
    pip install .
```
